A plugin for adding custom Gutenberg blocks to a theme.

== Description ==

Provides a collection of custom Gutenberg blocks including:

* Accordion with collapsible items
* Responsive image slider
* Image and text layout block
* Dynamic images with responsive sources
* Interactive map block (Leaflet)
* News slider (latest posts)
* Meeting list
* Section wrapper block
* PDF embed block

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Insert any of the custom blocks in the Gutenberg editor.
