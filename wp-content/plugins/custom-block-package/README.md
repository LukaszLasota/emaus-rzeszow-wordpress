# Custom Block Package

A collection of custom Gutenberg blocks for WordPress, built with `@wordpress/scripts`. Each block uses server-side rendering (`render.php`) and registers via `block.json`.

## Plugin Info

- **Version:** 1.0.0
- **Author:** Lukasz Lasota
- **Requires PHP:** 7.2+
- **Requires WordPress:** 5.9+
- **License:** GPL-2.0-or-later

## Architecture

```
custom-block-package/
├── index.php                  # Plugin entry: autoloader, RegisterBlocks, AssetsManager, cache hooks
├── app/
│   ├── Autoloader.php         # PSR-4 autoloader (CustomBlockPackage namespace)
│   ├── Blocks/
│   │   └── RegisterBlocks.php # Auto-discovers blocks from build/blocks/, registers category
│   └── Assets/
│       └── AssetsManager.php  # Registers Leaflet CSS + Glide.js CSS/JS libraries
├── src/blocks/                # Block source (JS, SCSS, PHP)
├── build/                     # Compiled blocks (generated by wp-scripts)
├── src/scss/                  # Shared styles (abstracts, variables, mixins)
├── src/js/
│   └── glide-init.js          # Glide carousel initialization utility
└── webpack.config.js          # Custom config extending wp-scripts
```

**Block category:** `custom-blocks-from-scratch` ("Custom blocks for sites")

**Cache invalidation:** Transient caches cleared automatically on post save:
- `save_post_post` → clears news slider cache
- `save_post_meetings` → clears meeting list cache

## Blocks

### PDF Block (`pdf-block`)

Embeds a PDF file with optional download button.

| Attribute | Type | Default | Description |
|-----------|------|---------|-------------|
| `pdfUrl` | string | — | PDF file URL |
| `pdfID` | number | — | Attachment ID |
| `pdfTitle` | string | — | PDF title |
| `btnTitle` | string | — | Download button text |
| `showDownload` | boolean | — | Show download button |
| `viewerHeight` | number | 0–1500 | Embed height (px) |
| `viewerWidth` | number | 20–100 | Embed width (%) |

Renders: `<embed>` element with optional download link.

---

### Dynamic Images (`dynamic-images`)

Responsive `<picture>` element with separate images for desktop, tablet, and mobile breakpoints.

| Attribute | Type | Description |
|-----------|------|-------------|
| `imgDesktopID/URL` | number/string | Desktop image |
| `imgTabletID/URL` | number/string | Tablet image (max-width: 768px) |
| `imgMobileID/URL` | number/string | Mobile image (max-width: 480px) |
| `heading` | string | Hidden H1 heading (SEO) |

Renders: `<picture>` with `<source>` media queries. Fallback `<img>` with `fetchpriority="high"` and `loading="eager"` (above-fold hero). Visually hidden `<h1>` for SEO.

---

### Responsive Image Slider (`responsive-image-slider`)

Image carousel using Glide.js with separate desktop/mobile image sets.

| Attribute | Type | Default | Description |
|-----------|------|---------|-------------|
| `desktopImages` | array | [] | Desktop images |
| `mobileImages` | array | [] | Mobile images |
| `mobileBreakpoint` | number | 767 | Breakpoint (px) |

Renders: Glide.js carousel with `<picture>` elements, navigation arrows, bullet pagination, pause/play button. Skips deleted attachments.

**Dependencies:** `@glidejs/glide`

---

### Emaus News Slider (`emaus-news-slider`)

Displays latest posts as a Glide.js carousel.

| Attribute | Type | Default | Description |
|-----------|------|---------|-------------|
| `numberOfPosts` | number | 3 | Posts to show (max 10) |
| `category` | string | — | Category filter |
| `autoplay` | boolean | true | Enable autoplay |
| `autoplaySpeed` | number | 3000 | Speed (ms, 1000–10000) |
| `linkDestination` | string | "post" | Link to "post" or "news_page" |
| `headingText` | string | "Aktualnosci" | Block heading |
| `perView` | number | 1 | Slides visible (1–3) |

Renders: Post cards with thumbnail, title, excerpt (20 words). **Cached** with 30-minute transient (key includes MD5 of attributes). Optimized query: `no_found_rows`, no meta/term cache.

**Dependencies:** `@glidejs/glide`

---

### Map Block (`map-block`)

Interactive Leaflet.js map with lazy-loaded tiles.

| Attribute | Type | Default | Description |
|-----------|------|---------|-------------|
| `latitude` | number | 50.031562 | Center latitude |
| `longitude` | number | 21.997937 | Center longitude |
| `zoom` | number | 16 | Zoom level |
| `containerHeight` | number | 300 | Height (px) |
| `popupText` | string | "Nasza lokalizacja" | Marker popup |

Renders: Map container with data attributes. Frontend JS initializes Leaflet with OpenStreetMap tiles, custom marker icon, and popup.

**Lazy loading:** IntersectionObserver with 600px rootMargin — tiles loaded only when map approaches viewport. Fallback for browsers without IO support.

**Dependencies:** `leaflet`

---

### Image Text (`image-text`)

Image with text overlay and optional link.

| Attribute | Type | Description |
|-----------|------|-------------|
| `text` | string | Caption text |
| `imgID` | number | Image attachment ID |
| `fullWidth` | boolean | Full width mode |
| `postURL` | object | Link URL + opensInNewTab |

Renders: `<figure>` with image, overlay div, and text caption. Optional `<a>` wrapper with `rel="noopener noreferrer"`.

---

### Section Block (`section-block`)

Flexible section container with grid/flex layout options.

| Attribute | Type | Default | Description |
|-----------|------|---------|-------------|
| `tagName` | string | "div" | HTML tag (div/section/article/main/header/footer/aside) |
| `backgroundImage` | string | — | Background image URL |
| `blockWidth` | string | — | Width (%, px, em, rem, vw, ch) |
| `layoutType` | string | "none" | Layout: none, grid, flex |
| `columnsMobile` | number | 1 | Grid columns on mobile |
| `columnsSmallTablet` | number | 2 | Grid columns on small tablet |
| `columnsLargeTablet` | number | 3 | Grid columns on large tablet |
| `columnsDesktop` | number | 4 | Grid columns on desktop |
| `gridGap` | number | 10 | Gap (px) |
| `flexDirection` | string | — | Flex direction |
| `flexWrap` | string | — | Flex wrap |

Renders: Dynamic HTML tag with CSS custom properties for responsive grid/flex. All CSS values validated against allowed lists.

**Supports:** Colors, typography, spacing, gradients.

---

### Custom Accordion (`custom-accordion`) + Accordion Item (`accordion-item`)

Parent-child accordion with smooth animations and keyboard navigation.

**Accordion Item attributes:**
- `title` (string) — Item heading

Renders: `<h3>` with `<button>` trigger, collapsible content panel. ARIA: `aria-expanded`, `aria-controls`, unique IDs via `wp_unique_id()`.

**Frontend JS:** Smooth max-height transitions, keyboard navigation (Arrow Up/Down, Home/End, Enter/Space).

**Supports:** Colors, typography, spacing, borders, dimensions.

---

### Meeting List (`meeting-custom-list`)

Displays meetings CPT as flipping cards.

| Attribute | Type | Default | Description |
|-----------|------|---------|-------------|
| `numberposts` | number | -1 | Posts to show (-1 = all) |
| `blockTitle` | string | "Nasze spotkania" | Heading |

Renders: Flipping cards (front: title, day/hour, place; back: post content). Sorted by ACF `priority` field. **Cached** with 30-minute transient. Cards are keyboard accessible (tabindex, role="button", aria-expanded).

**ACF fields used:** `day_hour`, `place`, `priority`

## Build

```bash
npm run build    # Production build (wp-scripts build)
npm start        # Watch mode (wp-scripts start)
```

### Custom Webpack Config

Extends `@wordpress/scripts` default:
- Sass loader with modern API
- `CopyWebpackPlugin` — copies map-block images and Leaflet CSS
- `IgnoreEmitPlugin` — prevents global SCSS from emitting JS
- Extra entry points: Glide.js assets, `editor-global.scss`, `frontend-global.scss`
