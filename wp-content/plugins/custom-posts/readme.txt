A plugin for registering custom post types and taxonomies.

== Description ==

Provides reusable builders for registering custom post types and taxonomies in WordPress:

* Meetings (Spotkania) post type with custom admin columns
* CptBuilder for easy CPT registration
* TaxBuilder for easy taxonomy registration

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
